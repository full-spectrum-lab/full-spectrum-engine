# GitHub Release Body: v1.0.0

## Full Spectrum Engine v1.0.0

`full-spectrum-engine v1.0.0` is the first sealed stable local-first contract release in the Full Spectrum engine line.

This release means:

- runnable locally
- reproducible with fixed seeds
- auditable through structured outputs
- explainable enough for review
- documented enough for outside developers to use without private handoff knowledge

It does **not** mean:

- production enterprise governance is complete
- protocol-network interoperability is complete
- final legal or business action should be delegated to the engine

### Included in this release

- deterministic simulation baseline
- committed golden samples
- structured error contract
- OpenAPI export
- API field and error-code docs
- explainability walkthrough
- warning governance
- release checklist and gate review
