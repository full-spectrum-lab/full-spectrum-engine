# 模块详解

> **作者：workbuddy** | 更新：2026-07-04

## Core (`src/core/`)

### state.py
- `CivilizationState(survival, coordination, meaning)` — 文明状态向量
- `is_feasible()` — 检查是否在悲悯约束集 C 内
- `project_to_feasible(state)` — 投影到可行域

### fshi.py
- `FSHIConfig(w_l=0.40, w_m=0.35, w_h=0.25)` — 权重配置
- `compute_fshi(state, config, penalty=0.0)` — 计算 FSHI (0-100)
- `fshi_risk_level(fshi)` — 风险分级

## Engine (`src/engine/`)

### ess.py
- `ESS(ESSConfig(horizon=5, num_candidates=10))` — 伦理觉性模拟器
- `select_strategy_with_result(S)` — 选择最优策略，返回 (W, ESSResult)
- 痛苦指数: `Pain = (1-S_l) + (1-S_m) + (1-S_h) + Penalty`

### agents.py
- 四类Agent: AIAgent / HumanAgent / RegulatorAgent / EnterpriseAgent
- `AuthorityMatrix` — 动态权威矩阵，根据系统状态自动调整权重
- `aggregate_strategies(agents, S, authority)` — 加权聚合策略

## Observation (`src/observation/`)

### l0.py
- `CompositeObservationOperator(sources, config)` — 复合观测算子
- 压缩映射: `L0(S) = prev + alpha * (avg - prev)`
- 观测源: FixedObservationSource / RandomObservationSource / LogBasedObservationSource

## Guardian (`src/guardian/`)

### network.py
- `GuardianNetwork()` — 守庙人网络
- 2/3 多签: 最少5人参与，获胜选项需2/3加权票数
- 类型: HUMAN / AI / INSTITUTIONAL

## Governance (`src/governance/`)

### validator.py
- `DreamButterflyValidator()` — 梦蝶四层校验
- L1 边界自知 (clarity < 0.3 拒绝)
- L3 成住坏空 (rigidity >= 0.8 拒绝)
- L4 能所纯度 (purity < 0.7 尝试修正)
- L5 觉性炸弹 (连续3次触发投影)

### bomb.py
- `AwarenessBombEngine()` — 觉性炸弹引擎
- 触发条件: 连续3次纯度不合格 / 刚性>0.85 / 生存危机
- 引爆: 投影到可行域 + 额外软化

## Bridge (`src/bridge/`)

### runestone.py
- `Runestone.create(decision, reason, risk_vector)` — 创建符石
- `RiskVector` — 八维语义风险向量
- `compute_hash()` — 链式验证哈希

## Safety (`src/safety/`)

### emergency.py
- `EmergencyBrakeEngine()` — 紧急制动 BSRM
- 自动触发: FSHI<15 / S_l<0.2 / 冲突>0.8 / 痛苦>0.9 / 连续炸弹>=3
- 复位需守庙人多签 (最少5人, 2/3通过)
