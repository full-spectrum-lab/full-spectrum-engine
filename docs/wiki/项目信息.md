# 项目信息

> **作者：workbuddy** | 更新：2026-07-04

## 项目定位

Full Spectrum Engine 是全频谱协议的**可运行仿真引擎层**，用于多 Agent 文明动力学模拟、相变检测与治理控制。

### 三层架构定位

| 层 | 仓库 | 定位 |
|---|---|---|
| 协议规范层 | `full-spectrum-ethics` | RFC、Schema、示例（"协议应该是什么样"） |
| **可运行引擎层** | **`full-spectrum-engine`（本项目）** | 仿真、推演、验证（"协议跑起来是什么样"） |
| 项目总控台 | `qpp` | Wiki、路线图、需求草稿、AI协作 |

## 设计承诺

1. **本地优先**：引擎默认不上传数据、不连接协议网络、不收集企业内容
2. **可断网运行**：所有仿真、验证、报告生成均可在完全离线环境下完成
3. **可验证**：每个决策路径都有完整的因果链记录和符石审计令牌

## 模块架构

| 模块 | 路径 | 功能 |
|------|------|------|
| Core | `src/core/` | 文明状态向量 + FSHI 频段健康指数 |
| Engine | `src/engine/` | ESS 伦理觉性模拟器 + 多 Agent 策略聚合 |
| Observation | `src/observation/` | L0 现实锚定层（复合观测算子、压缩映射） |
| Guardian | `src/guardian/` | 守庙人网络（2/3 多签投票）+ Lyapunov 调节器 |
| Governance | `src/governance/` | 梦蝶四层校验器 + 觉性炸弹引擎 |
| Bridge | `src/bridge/` | 符石审计令牌 + 八维语义风险向量 |
| Safety | `src/safety/` | 紧急制动系统 BSRM |
| Report | `src/report_generator.py` | 因果链报告生成器 |

## 核心概念

### 文明状态向量 S(t) = [S_l, S_m, S_h]^T

| 分量 | 名称 | 范围 |
|------|------|------|
| S_l | 生存层 (Survival) | [0, 1] |
| S_m | 协调层 (Coordination) | [0, 1] |
| S_h | 意义层 (Meaning) | [0, 1] |

### FSHI 频段健康指数

`FSHI = w_l * S_l + w_m * S_m + w_h * S_h - Penalty` (映射到 0-100)

| 等级 | FSHI | 含义 |
|------|------|------|
| EXCELLENT | >= 80 | 优秀 |
| NORMAL | >= 60 | 正常 |
| WARNING | >= 40 | 警告 |
| CRISIS | >= 20 | 危机 |
| CRITICAL | < 20 | 临界 |

### 悲悯约束集

`C = { S in [0,1]^3 | S_l >= 0.3, S_m <= 0.8, S_h >= 0.2 }`
