## 扩展引擎

### 添加新 Agent

```python
class CustomAgent(Agent):
    def policy(self, S: CivilizationState) -> np.ndarray:
        return np.array([0.3, 0.3, 0.4])

    def get_name(self) -> str:
        return "Custom_Agent"

    def get_type(self) -> str:
        return "Custom"
```

### 添加新观测源

```python
class CustomObservationSource(ObservationSource):
    def observe(self) -> CivilizationState:
        return CivilizationState(
            survival=self._read_survival(),
            coordination=self._read_coordination(),
            meaning=self._read_meaning()
        )

    def get_source_name(self) -> str:
        return "CustomSource"
```
